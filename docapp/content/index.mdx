## System Architecture

The platform consists of several microservices:

- **Agent**: The core trading bot that executes strategies.
- **Model**: Machine learning models for price prediction and sentiment analysis.
- **ExData**: External data collectors (Yahoo Finance, Reddit, etc.).
- **RtData**: Real-time data processing pipeline.
- **WebApp**: User interface for monitoring and control.
